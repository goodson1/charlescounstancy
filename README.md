# Charles Counstancy - Professional Services Website

A modern, responsive website template for professional services and portfolio showcase.

## Features

- **Responsive Design** - Mobile-first approach works on all devices
- **Modern Navigation** - Clean header with sticky navigation menu
- **Hero Section** - Eye-catching landing area with call-to-action
- **Services Grid** - Showcase 6 core service offerings with icons
- **Portfolio Section** - Display 6 project highlights with descriptions
- **Contact Section** - Easy client engagement with email call-to-action
- **Professional Footer** - Social links and copyright
- **Smooth Scrolling** - Seamless navigation experience
- **Accessibility** - WCAG compliant with keyboard navigation

## Project Structure

## Getting Started

1. **Open locally**: Double-click `index.html` or open in your browser
2. **Customize content**: Edit service descriptions, project titles
3. **Brand it**: Update colors in `style.css` (CSS variables in `:root`)
4. **Add images**: Replace portfolio placeholders with actual screenshot images
5. **Update contact**: Change email address in Contact section

## Customization

### Colors
Edit CSS variables in `style.css`:
```css
:root {
  --primary: #2563eb;      /* Blue */
  --secondary: #1e40af;    /* Dark Blue */
  --accent: #f59e0b;       /* Amber */
  --dark: #1f2937;         /* Dark Gray */
  --text: #374151;         /* Text Gray */
}
```

### Sections
- **Navigation**: Edit `.navbar` and `.nav-menu`
- **Hero**: Update `.hero` and `.hero-content`
- **Services**: Modify `.card` elements and icons
- **Portfolio**: Replace portfolio image placeholders with actual images
- **Contact**: Update email in contact section

### Typography
- Main font: System sans-serif stack
- Headings: 700 weight (bold)
- Body text: 400 weight (regular)

## Integration with Screenshots

The portfolio section includes 6 placeholder areas for your screenshot images:
- Replace "Screenshot 1-6" text with actual images
- Update project names (Alpha, Beta, Gamma, Delta, Epsilon, Zeta)
- Add descriptions for each project
